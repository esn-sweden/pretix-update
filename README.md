# pretix-update

A script to automate updates of Pretix

## Installation

Clone the repository

```sh
git clone https://github.com/esn-sweden/pretix-update
cd pretix-update
```

Make the script executable

```sh
chmod +x update_pretix.sh
```

Install to system path

```sh
sudo ln -s $(pwd)/update_pretix.sh /usr/local/bin/pretix-update
```

## Usage

```sh
sudo pretix-update
```
